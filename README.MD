# hello-umesh
Simple Hello World Node Package

## Installation
Install via NPM:

```bash
npm install hello-umesh

```

## Usage

#### javascript

```javascript

var myApp = require("hello-umesh");

myApp.printMsg(); // print and return " Hello Umesh! " 

```

#### TypeScript

```typescript

import * as myApp from 'hello-umesh';

myApp.printMsg(); // print and return " Hello Umesh! " 


```
